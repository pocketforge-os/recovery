# PocketForge Recovery

FEL unbrick and flashing host tooling for the TrimUI Smart Pro (Allwinner A133P, USB ID `1f3a:efe8`).

FEL is a BROM-level USB recovery mode that survives any corrupted bootloader. This repo holds:

- `sunxi-fel` wrapper scripts for re-imaging from a host machine
- Stock-restore procedures
- SD card re-flash helpers

The safety net that makes PocketForge's SD-boot approach zero-risk.

Populated in **Phase 1**. See the [pocketforge-os](https://github.com/pocketforge-os) org for the full repo set.
